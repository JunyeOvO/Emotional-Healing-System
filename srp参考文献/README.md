## 获取建议

1. 优先获取 Goessl 2017 和 Carl 2019 — 这是系统心理学依据最核心的两篇元分析
2. Bandura 1977 可从多个大学网站免费下载（经典文献，扫描版众多）
3. 其他缺失文献可通过 sci-hub.se 或学校图书馆数据库获取
4. 部分书籍（Lakoff 1980, Picard 1997）可在学校图书馆借阅纸质版
