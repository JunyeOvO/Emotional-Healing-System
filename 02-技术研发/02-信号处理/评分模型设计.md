# 评分模型设计

> 呼吸波形 → 特征提取 → breath_score + calm_index → UDP JSON

## 信号处理链

```
ECG (130Hz) ──→ 带通滤波(0.5-40Hz) ──→ R峰检测 ──→ HR + HRV(RMSSD)
RSP (25Hz)  ──→ 带通滤波(0.1-0.5Hz) ──→ 呼吸率(RR) + 呼吸深度 + 呼吸规律性
```

## 评分维度

| 指标 | 来源 | 含义 | 权重 |
|------|------|------|:----:|
| breath_score | RSP波形 | 呼吸与引导圈同步程度 (0-100) | 40% |
| hr_coherence | ECG | 心率变异性一致性 | 30% |
| stability | RR趋势 | 呼吸频率稳定性 | 20% |
| depth | RSP振幅 | 呼吸深度 | 10% |

## calm_index 计算

```
calm_index = breath_score × 0.4 + hr_coherence × 0.3 + stability × 0.2 + depth × 0.1
```

输出范围：0-100，映射到天气强度（100=晴天）

## 实时输出 (UDP JSON, 10Hz)

```json
{
  "ts": 1716300000.123,
  "breath_score": 72.5,
  "calm_index": 68.3,
  "hr": 72,
  "rr": 14.2,
  "phase": "inhale",
  "weather_intensity": 0.32
}
```
