# 设备方案与数据采集

> 呼吸胸带 + 心电胸带 → Python BLE → 原始信号 → CSV

## 设备选型

| 设备 | 型号 | 信号 | 采样率 | 接口 | 状态 |
|------|------|------|:------:|------|:----:|
| 心电胸带 | Polar H10 | ECG, HR, HRV, RR | 130Hz ECG | BLE SDK | 待采购 |
| 呼吸胸带 | 待选型 | RSP波形, RR | ≥25Hz | BLE/串口 | 待选型 |
| 皮电手环 | 辅助 | EDA, ACC, TEMP | ≥4Hz | BLE | 可选 |

## 数据流

```
Polar H10 ──BLE──→ Python (bleson/bleak)
                         │
                    NeuroKit2 处理
                         │
                    ┌────┴────┐
                    ↓         ↓
                 CSV日志    UDP JSON (10Hz)
                 (离线)     → TD / Unity
```

## 开发策略

1. **先用模拟数据跑通全链路**（Python 生成正弦波模拟呼吸 + ECG）
2. 真实设备到位后替换输入层
3. 模拟数据脚本放 02-技术研发/01-数据采集/
