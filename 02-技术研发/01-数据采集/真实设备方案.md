## 1. 架构总览

```
┌─────────────────────────────────────────────────────────────────────┐
│                         DEVICE LAYER                                 │
│  Polar H10          PLUX呼吸带         EDA腕带        (可选)         │
│  BLE 130Hz ECG      BLE/串口 ≥25Hz     BLE ≥4Hz                     │
│  ─── asyncio ───    ─── asyncio ───    ─── asyncio ───              │
│       ↓                  ↓                  ↓                       │
│  ecg_ringbuf        resp_ringbuf       eda_ringbuf     acc_ringbuf  │
│  (deque 2000)       (deque 500)        (deque 200)     (deque 200)  │
└────────────┬────────────────────────────────────────────────────────┘
             │  read_latest() @ 10Hz tick
             ▼
┌─────────────────────────────────────────────────────────────────────┐
│                     FRAME ASSEMBLER (10Hz clock)                     │
│  time-align → downsample → unified RawFrame → queue.Queue(100)      │
└────────────┬────────────────────────────────────────────────────────┘
             │  queue.get() in main thread
             ▼
┌─────────────────────────────────────────────────────────────────────┐
│                     SIGNAL PIPELINE (main thread)                    │
│  RawFrame → SignalPipeline.feed() → ProcessedFrame                  │
│          → ScoringModel.score()    → ScoreFrame                     │
│          → UDPSender.send()        → TD:5005 / Unity:5006           │
│          → CSVLogger.write()       → 实验数据/                       │
└─────────────────────────────────────────────────────────────────────┘
```

**核心原则：**
- 每个传感器独立线程采集，互相不阻塞
- 10Hz 帧组装器从各 ring buffer 读取最新值，时间对齐
- 主线程保持同步，只管消费 queue 中的组装帧
- 任一设备缺失 → 对应通道填 0，管道优雅降级

### 2.1 为什么 BLE 用独立 asyncio 线程

Polar H10 的 `bleak` 库基于 `asyncio`。Python 的 `asyncio` 不能跨线程共享 event loop，所以每个设备有两种选择：

**方案 A（推荐）：单 asyncio 线程 + 多设备**
```
一个 daemon 线程跑一个 asyncio event loop
  ├── bleak.BleakScanner 发现设备
  ├── PolarH10Client(bleak.BleakClient) → ecg_callback → ecg_buf
  ├── RespirationBeltClient             → resp_callback → resp_buf
  └── EDAClient                         → eda_callback → eda_buf
```
优点：资源开销小，线程数少。缺点：一个设备断连可能影响 loop 稳定性。

**方案 B：每设备独立 asyncio 线程**
```
daemon thread 1: asyncio loop 1 → Polar H10
daemon thread 2: asyncio loop 2 → PLUX Belt
daemon thread 3: asyncio loop 3 → EDA Wrist
```
优点：设备完全隔离，一个设备故障不影响其他。缺点：3x 线程开销。

**采用方案 A** — 设备数 ≤3，单 loop 足够。

### 2.2 帧时钟（10Hz）

独立的 `threading.Timer` 或 `while not stopped: sleep(0.1)` 线程：

```python
class FrameClock(threading.Thread):
    def run(self):
        while not self._stop:
            ts = time.time()
            frame = RawFrame(
                timestamp=ts,
                respiration_raw=resp_buf.read_latest(),
                ecg_raw=ecg_buf.read_latest(),
                eda_raw=eda_buf.read_latest(),
                acc_magnitude=acc_buf.read_latest(),
                temp_skin=temp_buf.read_latest(),
            )
            self.output_queue.put(frame, timeout=1.0)
            elapsed = time.time() - ts
            time.sleep(max(0, 0.1 - elapsed))
```

### 3.2 环形缓冲区（线程安全）

```python
import threading
from collections import deque

class RingBuffer:
    """Thread-safe ring buffer for sensor samples with timestamp."""

    def __init__(self, capacity: int):
        self._buf: deque[tuple[float, float]] = deque(maxlen=capacity)
        self._lock = threading.Lock()

    def push(self, timestamp: float, value: float) -> None:
        with self._lock:
            self._buf.append((timestamp, value))

    def read_latest(self) -> float:
        """Return most recent value, or 0.0 if empty."""
        with self._lock:
            if self._buf:
                return self._buf[-1][1]
            return 0.0

    def read_window(self, n: int) -> list[float]:
        """Return last n values for downsampling."""
        with self._lock:
            return [v for _, v in list(self._buf)[-n:]]

    def __len__(self) -> int:
        with self._lock:
            return len(self._buf)
```

## 5. 帧组装器

### 5.1 时间对齐策略

各设备采样率不同（130Hz / 25Hz / 4Hz），统一到 10Hz：

```
10Hz tick
  ├── ecg_raw   = mean(ecg_buf.read_window(13))   # 130→10Hz
  ├── resp_raw  = mean(resp_buf.read_window(3))    # 25→10Hz (近似)
  ├── eda_raw   = eda_buf.read_latest()            # 4Hz→最近值
  ├── acc_mag   = acc_buf.read_latest()
  └── temp_skin = temp_buf.read_latest()
```

## 6. DeviceManager — 设备编排器

统一管理所有设备的生命周期：

```python
class DeviceManager:
    """Start/stop/monitor all BLE devices. Single asyncio loop."""

    def __init__(self):
        self.drivers: dict[str, DeviceDriver] = {}
        self.buffers: dict[str, RingBuffer] = {}
        self.frame_clock: FrameClock | None = None
        self._loop: asyncio.AbstractEventLoop | None = None
        self._thread: threading.Thread | None = None

    def register(self, name: str, driver: DeviceDriver, buffer: RingBuffer):
        self.drivers[name] = driver
        self.buffers[name] = buffer

    def start(self) -> bool:
        """Start BLE event loop in daemon thread, connect all devices."""
        self._thread = threading.Thread(target=self._run_loop, daemon=True)
        self._thread.start()
        # Wait for connections (with timeout)
        # Start frame clock
        self.frame_clock = FrameClock(self.buffers)
        self.frame_clock.start()
        return True

    def _run_loop(self):
        self._loop = asyncio.new_event_loop()
        asyncio.set_event_loop(self._loop)
        self._loop.run_until_complete(self._connect_all())

    def stop(self):
        if self.frame_clock:
            self.frame_clock.stop()
        # Disconnect all drivers
        # Stop event loop
```
