# Unity 2D 横板像素场景设计（PC 桌面端）

> 四种天气场景 + 旅人像素动画 + 粒子特效 + 恢复过渡
> **呈现方式：PC 显示器全屏 | 视觉风格：2D 横板像素化**

## 场景架构

```
Unity Project/
├── Scenes/
│   ├── StormScene.unity    风暴（焦虑）
│   ├── HeatScene.unity     炙烤（烦躁）
│   ├── SnowScene.unity     暴雪（低落）
│   └── FadeScene.unity     褪色（孤独）
├── Scripts/
│   ├── OSCReceiver.cs      UDP接收
│   ├── WeatherController.cs 天气强度→视觉效果映射
│   ├── TravelerAnim.cs     旅人Sprite状态机
│   └── TransitionManager.cs 天气→晴天过渡
└── Assets/
    ├── Sprites/            旅人+场景像素Sprite
    ├── Animations/         Animator Controller
    └── Particles/          雨/火/雪/光点粒子Sprite
```

## 旅人状态机

```
普通行走 → 暴风雨中缓慢 → 保护罩出现 → 保护罩修复 → 正常行走
   ↑                                                       ↓
   └──────────── 天空放晴 ← 天气减弱 ← 呼吸同步 ←──────────┘
```

对应 40-srp-animation-state-machine。

## 天气强度映射

| calm_index | weather_intensity | 视觉效果 |
|:----------:|:-----------------:|----------|
| 0-25 | 1.0 | 全强度暴风雨/炙烤/暴雪/褪色 |
| 25-50 | 0.7 | 中等强度，可见旅人 |
| 50-75 | 0.4 | 弱化，天空开始放晴 |
| 75-100 | 0.1 | 接近晴天，轻风微云 |
