## 1. 项目概述

### 一句话说明

> 把大学生常见的焦虑、烦躁、低落、孤独四类情绪体验转化为四种天气场景，通过呼吸胸带和心电胸带监测身体状态，让用户通过呼吸训练看到天气逐渐恢复，从而学习可操作的情绪调节方法。

### 核心体验

旅人在恶劣天气中前行；用户通过呼吸训练帮助天气恢复、旅人继续前进。

### 项目定位

| 维度 | 定位 |
|------|------|
| 项目类型 | 生理数据驱动的情绪疗愈与教育交互系统 |
| 面向对象 | 普通大学生，不面向临床诊断人群 |
| 教育目标 | 让学生理解情绪像天气一样会变化，可通过呼吸、觉察和微行动逐步恢复 |
| 技术边界 | 情绪调节教育体验研究，不做医学诊断，不宣称治疗 |

### 四种情绪天气

| 场景 | 情绪 | 隐喻 | 呼吸策略 | 恢复信号 |
|------|------|------|----------|----------|
| 风暴 | 焦虑 | 混乱、失控 | 4-2-6 呼吸 | 保护罩修复、风雨减弱 |
| 炙烤 | 烦躁 | 闷热、急躁 | 长呼气降温 | 热浪消退、冷色气流 |
| 暴雪 | 低落 | 冻结、停滞 | 稳定呼吸+微行动 | 脚印显现、雪势减弱 |
| 褪色 | 孤独 | 连接感下降 | 自主呼吸+感官连接 | 世界重新上色 |

### 六子系统划分

| # | 子系统 | 一句话解释 | 负责人 |
|---|--------|-----------|--------|
| 1 | 情绪天气体验 | 把4种情绪变成4种天气（Unity 全权渲染） | 数字媒体 + 交互设计 |
| 2 | 生理数据采集 | 用胸带采集呼吸与心跳 | SE |
| 3 | 数据处理与建模 | 把设备读数翻译成评分 | SE |
| 4 | TouchDesigner 渲染+监控 | 呼吸引导圈渲染(Spout推送)+实验监控台+数据曲线 | 交互设计 |
| 5 | Unity 沉浸视觉 | Spout纹理展示+天气场景+旅人+粒子+提示词（用户所见） | 数字媒体 |
| 6 | 实验评价与成果转化 | 问卷/论文/专利/软著 | 全员 |

### 数据流

```
设备 → Python → UDP JSON (10Hz) → :5005 → TouchDesigner (渲染呼吸圈 + 主试监控)
                                 → :5006 → Unity (天气场景 + 展示呼吸圈纹理)
        │                       TD → Spout → Unity (呼吸圈纹理推送)
        └→ CSV 日志 (离线分析)
```

## 6. 技术栈

| 层级 | 技术 | 用途 |
|------|------|------|
| 数据采集 | Python BLE / Polar SDK | 连接呼吸胸带和心电胸带 |
| 信号处理 | Python + NeuroKit2 + BioSPPy | ECG/RSP滤波、特征提取 |
| 数据建模 | Python + NumPy/SciPy | 呼吸评分、平静指数计算 |
| 通信 | UDP JSON (10Hz) | Python → TD / Unity 实时传输 |
| 实时可视化 | TouchDesigner | 呼吸引导圈渲染(Spout推送)、实验监控台、实时数据曲线 |
| 2D像素渲染 | Unity 2D | Spout纹理展示、天气场景、旅人动画、粒子特效、环境提示词（用户所见） |
| 纹理共享 | Spout (Windows) | TD Spout Out TOP → Unity KlakSpout Receiver |
| 实验分析 | Python + Pandas + Matplotlib | 数据统计与可视化 |
| 版本控制 | Git + GitHub | 代码与文档版本管理 |

## 8. 风险管理

| # | 风险 | 概率 | 影响 | 应对策略 |
|---|------|:----:|:----:|----------|
| R1 | 真实设备到位晚或不稳定 | 中 | 高 | 先用Python模拟数据跑通TD和Unity全链路；真实设备接入作为替换输入层 |
| R2 | 技术链路理解困难 | 高 | 中 | 使用类比法表达（设备=尺子、Python=翻译器、TD=控制台、Unity=舞台） |
| R3 | 场景做得过大 | 中 | 高 | 每种天气只做一个核心视觉机制，不扩展多场景地图 |
| R4 | 提示词变成说教 | 低 | 中 | 提示词从环境自然出现（雨幕/热浪/雪雾/褪色），不用角色说教 |
| R5 | 心理健康表述过度 | 中 | 高 | 全程使用"情绪调节教育""日常情绪困境"，不使用治疗宣称 |
| R6 | 多系统同步问题 | 中 | 中 | UDP JSON统一协议，TD(:5005)和Unity(:5006)各收独立端口，互不干扰 |
