## 4. 实验设计

### 基础参数

| 参数 | 值 | 为什么 |
|------|-----|--------|
| 设计类型 | 被试内设计 | 被试少(8-12)，个体差异可控，每人体验全部4天气 |
| 被试人数 | 8-12 名大学生 | 非严肃判断人群，小样本可用性测试够用 |
| 天气顺序 | 随机(Latin Square) | 消除顺序效应 |
| 单人时长 | 约40min | 5min介绍 + 4×7min体验 + 5min访谈 |
| 采集数据 | 生理数据(全程) + 问卷(体验后) + 访谈(体验后) | 定量+定性三角验证 |

### 实验流程

```
1. 介绍(5min)
   ├── 签署知情同意书
   ├── 佩戴设备(Polar H10 + 呼吸胸带)
   └── 填写 SAM 前测

2. 体验(4×7min = 28min)
   ├── 天气1 → SAM (体验后)
   ├── 天气2 → SAM
   ├── 天气3 → SAM
   └── 天气4 → SAM + 自编问卷

3. 访谈(5min)
   └── 半结构化访谈

4. 结束(2min)
   └── 取下设备、道谢
```

## 5. 测量工具

### SAM 量表（Self-Assessment Manikin）

| 维度 | 5点图示 | 时机 |
|------|---------|------|
| 效价 (Valence) | 😊—😐—😞 | 体验前后各一次 |
| 唤醒度 (Arousal) | 😲—😐—😴 | 体验前后各一次 |

### 自编体验问卷（5点 Likert）

1. 我感觉自己沉浸在天气场景中（1=完全不同意, 5=完全同意）
2. 呼吸引导圈很容易跟上（1=完全不同意, 5=完全同意）
3. 提示词的出现方式让我感到自然（1=完全不同意, 5=完全同意）
4. 体验后我感觉比之前平静（1=完全不同意, 5=完全同意）
5. 我愿意再次使用这个系统（1=完全不同意, 5=完全同意）

### 半结构化访谈提纲

1. 你在哪个天气场景中感受最深？为什么？
2. 呼吸引导圈好不好跟？有没有让你分心的时候？
3. 旅人在场景中的存在让你有什么感觉？
4. 体验结束后，你对"情绪像天气"这个比喻的理解有变化吗？
5. 有什么让你不舒服或困惑的地方吗？
6. 你会怎么向朋友描述这个体验？
