## 1. 项目范围说明书

### 1.1 项目边界

| 维度 | 在范围内 | 超出范围 |
|------|----------|----------|
| **功能** | 4 种情绪天气场景、呼吸引导圈、实时数据监控台、UDP 数据桥接、CSV 实验日志 | 移动端 App、VR/AR 模式、多人同时交互 |
| **技术** | Polar H10 BLE + 呼吸胸带 → Python → UDP → TD/Unity | 第三方可穿戴设备协议、云端推理、联网功能 |
| **交付** | 桌面端可运行系统、8-12 人实验数据、论文稿件、答辩材料 | 医疗器械认证、临床有效性验证、产品化部署 |
| **用户** | 普通大学生（非临床人群） | 临床诊断患者、未成年人、老年人 |

### 1.2 产品定义

```
SRP v1.0 = 实时生理数据采集 + 8 维情绪评分模型 + 4 天气视觉场景 + 呼吸训练交互 + 实验数据记录
```

### 1.3 验收标准 (Overall Acceptance Criteria)

| # | 标准 | 测量方式 |
|:--:|------|----------|
| AC-1 | 端到端数据流延迟 < 200ms (Python→TD 渲染) | UDP 时间戳差值 |
| AC-2 | 四种天气场景完整可体验，含呼吸引导和天气过渡 | 人工走查 |
| AC-3 | 评分模型 8 维指标全部输出，与 mock data 信号逻辑一致 | 单元测试 |
| AC-4 | 实验采集 8 人以上完整数据，CSV 格式符合标注规范 | 数据审核 |
| AC-5 | 文档全程使用「交互状态估计」等非医学术语 | 术语扫描 |
| AC-6 | 系统在 600 帧 (60s) 持续运行零崩溃 | 压力测试 |

## 4. 技术架构决策

### 4.1 架构决策记录 (ADR)

| ADR# | 决策 | 理由 | 日期 |
|------|------|------|------|
| ADR-1 | 评分模型独立于天气场景 | 8 维评分对所有天气通用，仅权重不同 | 5/22 |
| ADR-2 | UDP 替代 OSC | JSON 可读性强，方便调试和 CSV 对齐 | 5/23 |
| ADR-3 | MCP 开发时 + UDP 运行时 | MCP 搭工程，UDP 传数据，两者互补 | 5/28 |
| ADR-4 | TD 负责交互引导, Unity 负责场景视觉 | 各有擅长领域，UDP 同时投喂两者 | 5/29 |
| ADR-5 | Mock Data 先行, 真机延迟接入 | 硬件采购周期不可控，mock 保证开发进度 | 5/22 |
| ADR-6 | Python 主循环 10Hz | 生理信号变化频段在 0.01-1Hz，10Hz 足够且不丢信息 | 5/24 |

## 6. 接口协议定义

### 6.1 数据流向

```
Polar H10 (ECG 130Hz) ──┐
                         ├── BLE ──► Python (10Hz downsample)
呼吸胸带 (RSP 25Hz) ────┘              │
                              ┌─────────┴─────────┐
                              │ SignalPipeline     │
                              │ → ScoringModel     │
                              │ → UDPSender        │
                              └─────────┬─────────┘
                                        │ UDP JSON @ 10Hz
                          ┌─────────────┼─────────────┐
                          ▼             ▼             ▼
                   TD (127.0.0.1:5005)  Unity (5006)  CSV Logger
```

### 6.2 接口契约

详见 `02-技术研发/05-通信协议/UDP协议定义.md`。核心字段：

| 字段路径 | 类型 | 频率 | 消费者 |
|----------|------|:--:|--------|
| `scores.*` (8 维) | float | 10Hz | TD 监控台 |
| `weather.type` | str | 10Hz | TD + Unity |
| `weather.intensity` | float 0-1 | 10Hz | Unity 粒子控制 |
| `breath.phase` | str | 10Hz | TD 引导圈 |
| `breath.rate` | float | 10Hz | TD 曲线 |
| `cardiac.hr` | float | 10Hz | TD 曲线 |
| `cardiac.rmssd` | float | 10Hz | TD 监控台 |
| `guidance.prompt` | str | 1Hz | TD 提示词 |
| `guidance.circle_radius` | float 0-1 | 10Hz | TD 引导圈缩放 |
