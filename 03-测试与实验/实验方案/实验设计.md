# 实验方案

> 小样本可用性测试 → 问卷/访谈 → 数据分析 → 系统迭代

## 实验设计

| 维度 | 方案 |
|------|------|
| 类型 | 被试内设计 (within-subject) |
| 被试 | 8-12名大学生（非临床人群） |
| 场景 | 每个被试体验全部4种天气（顺序随机） |
| 时长 | 每人约40分钟（5min介绍 + 4×7min体验 + 5min访谈） |
| 采集 | 生理数据（全程）+ 体验后问卷 + 半结构化访谈 |

## 测量工具

| 工具 | 测量维度 | 时机 |
|------|----------|------|
| SAM量表 | 效价/唤醒度 | 体验前后 |
| 自编体验问卷 | 沉浸感/易懂性/舒适度 | 体验后 |
| 半结构化访谈 | 定性反馈 | 体验后 |
| 生理数据 | breath_score/calm_index 变化趋势 | 体验中 |

## 数据标注规范

参考 40-srp-reporting-standard：
- 被试编号匿名化（P01-P12）
- 数据文件命名：`P{ID}_{天气}_{日期}.csv`
- 图表标注：采样率、滤波参数、被试数
- 禁止医疗表述
